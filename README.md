### An unofficial, remote-play, adaptation of the card game Monikers, written in Socket.IO and Vue.js

### Check it out online at:

[playmonikers.com](https://www.playmonikers.com)


### To run locally:

```
npm install
npm run build
npm run start
```
Go to `localhost:3000`
